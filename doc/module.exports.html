exports

exports

Constructs a quadtree for quickly finding points in a given zone: O(log n) Inspired from https://github.com/CodingTrain/QuadTree There are tow modes: *normalMode* and *indexMode* - In normalMode, the methods deal directly with points (an array with 2 values). ie, when using 'getPoints', the method will return the points themselves - In indexMode, the methods deal with point indices: references to points within a dataset To use indexmode, use the 'setData' method. This will provide the data to map indices to, and construct the quadtree from it
